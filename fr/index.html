CatchChallenger


Ce jeu est un MMORPG, jeu Solo ou Lan. Vous pouvez combattre, farmer, explorer, crafter, échanger, commercer, faire de la gestion, de la competition... et vous être libre de jouer dans le style qui vous conviens.

C'est un jeu en pixel art game de type old school. Le travail est concentré sur le gameplay/performance/creativité. Les revenus sont pour payer le développeur et la partie artistique. Le jeu est complétement open source (GPL3).

Le jeu n'as pas de partie temps réél. Ce qui donne un avantage claire sur les connexion lente (3G/wifi) et du tiers monde. Vous pouvez y jouer en PK, PvP, TvT.
