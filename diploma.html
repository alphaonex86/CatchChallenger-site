Diploma



The diploma can be changed at anytime. The diploma is teorical and practical and need be validated by a large project in open source software

  • Bronze

    • OS and scheduling: Be able to understand the different OS type, the role of OS, their different layer, understand the syscall. The different type of scheduling for the different type of device and how correctly use it
    • Networking: Understand the different mode of transmition, the routing, round-trip delay time, network congestion, load balancing, high availability
    • Multi OS: Make generic code for all OS but be able to use specific OS/lib function when useful
    • C10K: The C10k problem is the problem of optimising network sockets to handle a large number of clients at the same time. The name C10k is a numeronym for concurrently handling ten thousand connections. High number of concurrent connections requires efficient scheduling of connections. Event Driven Programming.
    • Data format/container: Known the common data format on disk (png, xml, ...) and their data format, the metadata and their usage (easy to edit, performance, space). Known the in memory format: table, B-tree, integer size, variables multiplexing
    • Databases: The SQL (RDSMS) and NoSQL, be able to do their own database server

  • Silver

    • Algorithmic: Master the algorithmic complexity, optimise the logic, prevent error
    • Compression: Choice the right compression depending of the usage (compression/decompression speed and ratio), calculate the improvement and worse case, restructure the data (order and format) to improve the ratio, determine the typical(s) data(s)
    • Cryptography: Use correctly the hash, salt, synmetric and asynmetric compression
    • Security: Known the most common threat in programming like buffer overflow, XSS, SQL injection, DDOS

  • Gold

    • Hardware conception: Be able to contruct logical door, memory, bus, routing, cache, understand it into modern hardware context with millions of transistors by chip. Understand the external chip communication, and communication on more than 50 meters, in order/out of order, L1/L2
    • 0 Copy/Register/SIMD: Be able to code with memory copy only when is useful, understand the register usage. Use correctly the SIMD and SISD.
    • Language/jit/compiler: Choice the correct language on the market depending of the need, understand the process of compilation (Intermediate representation and their optimisation, specific hardware optimisation), jit and all the virtual machine related

  • Diamond

    • C10M: Be able to support 10 millions of connected client by node with good packets by seconds flow. Need understand the cache/buffer at software level. Blocking/not blocking IO.
    • Exascale: Working cluster with millions of nodes with good efficiency, very large scale operation, routing, replication, failover
